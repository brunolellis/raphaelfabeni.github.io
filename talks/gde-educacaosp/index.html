Guia de Estilos depois de projetos já desenvolvidos

Dificuldades

  • Códigos já desenvolvidos
  • Projetos já em produção e/ou fase de testes
  • Alta complexidade e margem de erro muito grande

Mudanças

  • Projetos já desenvolvidos utilizariam a versão antiga do Guia de Estilo
  • Novos projetos deveriam utilizar uma nova versão do Guia, que deveria servir como base para o desenvolvimento

<!doctype html>
<!--[if IE 7]>    <html class="no-js ie7" lang="pt-br">     <![endif]-->
<!--[if IE 8]>    <html class="no-js ie8" lang="pt-br">     <![endif]-->
<!--[if IE 9]>    <html class="no-js ie9" lang="pt-br">     <![endif]-->
<head>
   <title>{% if page.title %} {{ page.title }} | {% endif %} Guia de Estilos - Educação SP</title>
   <!-- Meta -->
   <meta charset="utf-8">
   {% include meta.html %}
   <!-- CSS -->
   {% include estilos.html %}
</head>
<body class="{{ page.body_class }} {{ page.page_type }}">

   <!-- Header -->
   {% include header.html %}

   <div class="container">
      {% include menu.html %}

      <article class="main" role="main">
         <h2 class="styleguide-title-page">{{ page.title }}</h2>

         {{ content }}

      </article>

   </div>

   <!-- Footer -->
   {% include footer.html %}

   <!-- Scripts -->
   {% include scripts.html %}
</body>
</html>
